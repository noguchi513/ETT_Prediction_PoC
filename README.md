# ETT データ予測PoC

このリポジトリは、ETTデータセットを用いた予測モデルの実装と、クライアント向けPoC報告スライドをまとめたものです。

## 概要
ETTデータセットの各センサーデータ（`h1`, `h2`, `m1`, `m2`）を対象に、短期および長期の予測モデルを実装しています。  
また、EDA（探索的データ解析）を行い、データの特徴や傾向を可視化しています。

## ファイル構成
```

ETT_Prediction_PoC/
├─ README.md               # このファイル
├─ requirements.txt        # 必要ライブラリ一覧
├─ PoC_Report.pdf          # クライアント向け報告スライド
├─ data_sample/            # サンプルデータ（任意）
├─ notebooks/              # 解析・予測用ノートブック
│   ├─ EDA.ipynb           # データの探索・可視化
│   ├─ h1.ipynb            # h1 データ用短期予測モデル
│   ├─ h2.ipynb            # h2 データ用短期予測モデル
│   ├─ m1.ipynb            # m1 データ用短期予測モデル
│   ├─ m2.ipynb            # m2 データ用短期予測モデル
│   └─ h1_long.ipynb       # h1 データ用長期予測モデル

````

## データについて
- 本リポジトリには、クライアントから提供された実データは含まれていません。
- 各ノートブックを実行するには、向こうから提供されたデータを `data/` フォルダに置いてください。
- サンプルデータやダミーデータは `data_sample/` フォルダに入れてあります。

## 使用方法
1. リポジトリをクローン
   ```bash
   git clone <リポジトリURL>
   cd <リポジトリ名>
````

2. 必要なライブラリをインストール

   ```bash
   pip install -r requirements.txt
   ```
3. 各ノートブックを順番に実行して予測結果を確認

   * `EDA.ipynb` → データ解析
   * `h1.ipynb` / `h2.ipynb` / `m1.ipynb` / `m2.ipynb` → 短期予測
   * `h1_long.ipynb` → 長期予測

## 必要環境

* Python 3.9+
* pandas, numpy, matplotlib, seaborn, scikit-learn など

## 成果物

* 各データセット用の予測モデルノートブック
